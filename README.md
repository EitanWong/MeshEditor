# 食用方法
- <b>①.Shift+M打开网格模型编辑器
  - (如果在Hierarchy视图已经选中GameObject，则会自动添加编辑项)
- <b>②.拖入GameObject，添加编辑项
- <b>③.选择对应功能进行编辑
